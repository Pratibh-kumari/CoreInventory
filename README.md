# CoreInventory

A backend inventory management system built with Node.js, Express, and PostgreSQL.

## Tech Stack
- Node.js + Express
- PostgreSQL (Neon)
- JWT Authentication
- Nodemailer (OTP)

## Setup

```bash
cd backend
npm install
# add your .env based on .env.example
npm run dev
```

## API Base URL
`http://localhost:5000/api`
